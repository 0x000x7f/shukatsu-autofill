# 就活エントリー自動入力

就活エントリーフォームに、事前登録した個人情報をワンクリックで自動入力する Chrome 拡張機能です。

## 対応プラットフォーム

| システム | 主な利用企業 |
|---|---|
| **i-webs.jp 系** | キオクシア・ルネサス・日立システムズ/ソリューションズ・NECネッツエスアイ・伊藤忠テクノ・NTTデータ（子会社）・野村総研・ソニー・三菱UFJ銀行・三井住友銀行・SCSK など |
| **axol.jp 系** | IBM Japan・TIS・信越化学・東芝・みずほFG・野村證券・KDDI・TDK・富士ソフト・NECグループ など |
| **e2r.jp 系** | 大和証券・三井住友信託銀行 |
| **snar.jp 系** | NTTデータ（本体）・NTTドコモ・NTT東日本・ソフトバンク など |

> **対応外**: 富士通（Workday ATSのためフィールドIDが動的生成）

## インストール

### 開発者モードで読み込む（推奨）

1. このリポジトリを ZIP ダウンロードまたは `git clone`
2. Chrome で `chrome://extensions/` を開く
3. 右上「デベロッパーモード」を ON
4. 「パッケージ化されていない拡張機能を読み込む」→ このフォルダを選択

## 使い方

### 1. 個人情報を登録する

拡張機能アイコンをクリック →「個人情報を編集」から以下を登録して「保存」。

- 氏名（漢字・カナ・ローマ字）
- 生年月日・性別
- 現住所・電話番号
- 休暇中の連絡先（現住所と異なる場合）
- ゼミ・サークル名
- 卒業年月
- メールアドレス

> データは **この PC 内（chrome.storage.local）にのみ保存** されます。外部送信・クラウド同期はしません。

### 2. エントリーフォームで自動入力する

対応サイトのエントリーフォームを開いた状態で拡張機能アイコンをクリック。  
「✓ 対象ページを検出」と表示されたら「自動入力を実行」ボタンを押す。

入力結果（成功件数 / スキップ件数）が表示されたら、内容を確認して送信してください。

## 自動入力される項目

- 氏名（漢字・カナ・ローマ字）
- 生年月日・性別
- 郵便番号・都道府県・住所・電話番号
- 休暇中の連絡先
- ゼミ・サークル名
- 卒業年月
- メールアドレス（確認欄も含む）

> **自動入力されない項目**: 大学名・学部・学科（各サイト独自のコードを使うため）、志望動機・自己PR（自由記述欄）

## プライバシー

入力した個人情報はお使いのPC内（`chrome.storage.local`）にのみ保存され、外部サーバーへの送信・第三者への提供・クラウド同期は一切行いません。

**プライバシーポリシー**: https://0x000x7f.github.io/shukatsu-autofill/privacy.html
